contain是用来在页面中动态include一个url的方法，类似于jsp中的jsp:include。

**jsp:include和contain是有区别的：include是将页面拼接好后之后在进行渲染，而contain是先渲染子页面再拼接，所以contain能够支持异步渲染。

基本用法
$contain.get('/widget/base.htm')

向页面传递参数
$contain.get('/widget/set_param.htm')

多个contain页面的响应问题--同步contain
$contain.get('/widget/block.htm')

bigpipe基本使用--异步contain
$contain.get('/widget/bigpipe.htm')
