配置文件说明和示例

系统使用配置文件进行全局配置，默认的配置文件为/WEB-INF/conf/server.properties. 通过配置web.xml文件可以设定制定的配置文件路径：

<context-param>
<param-name>melodyConfigLocation</param-name>
<param-name>/WEB-INF/conf/play.properties</param-name>
<context-param>

我们这里制定了play.properties为默认的配置文件路径。 配置文件中至少配置如下参数即可启动一个项目：

app.server.host=127.0.0,1
app.server.port=8081
melody.acyion.scan=com.eye.melody.demo.web
system.load.charset=UTF-8
web.encoding=GBK
