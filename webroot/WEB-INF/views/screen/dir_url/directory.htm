页面目录结构介绍：

页面文件放置于/WEB-INF/views/中，该文件夹下有layout和screen两个文件夹：

  • layout文件夹：用于放置模板文件
  • screen文件夹：用于放置页面内容文件

注意事项:

模板文件可以为多个页面提供公用的模板内容，并将页面文件嵌套其中，关于模板文件的使用有以下几点需要注意：

  • layout文件夹下的文件和文件夹与screen中的文件及文件夹有对应关系的，如layout/login/中的模板文件不会对screen/login/之外的页面文件生效。
  • default.vm模板文件会对所在文件夹及其子文件夹对应的所有页面文件生效，并且覆盖父文件夹中的default.vm的效果
  • 非default的模板文件会对其对应位置为同名页面文件生效，且覆盖default文件的效果
  • 页面文件需要有对应的模板文件，一般会在layout/下放置default.vm文件以保证每个页面都能正确显示。
