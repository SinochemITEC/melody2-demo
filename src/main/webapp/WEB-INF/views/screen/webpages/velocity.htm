Velocity是Apache基金会项目中一个模板语言项目，可以很好的简化页面开发的过程。

基本语法：

#set($foo = 'text')
  • 变量：

    • Velocity的变量由$+{变量名}组成，如$foo = ${foo}。

    需要注意的是，变量在已经被赋值时会自动替换成其数值，但是变量为空时会显示为变量的代码本身或抛出异常, 如需解决这个问题可以在$后增加'!'，这样在变量为空值时可以便不会显示任何内容。

    另外，虽然不需要大括号也能显示变量，但是为保证代码美观和易读性、减少错误，建议使用大括号。

  • 赋值：

    • Veloctiy中的赋值语句为：#set($<变量名> = <值> )，如#set($foo = 'new value')。

  • 成员变量：

    • Controller层返回的参数是对象，并且含有成员变量时，可以直接获取其成员变量的值，如${foo.value}。

    注意：当对象中有public类型的同名变量时会直接返回该变量值，否则会返回该变量的get方法的返回值。

  • 成员方法：

    • Controller层返回的参数是对象，并且含有成员方法时，可以直接获取其成员方法的返回值，如${foo.getValue()}。

    另外，如果方法需要参数，可以直接在调用方法时加入参数，如$page.setTitle( "My Home Page" )，$person.setAttributes( ["Strange", "Weird", "Excited"] )

  • 判断语句：

    • Velocity的判断语句是if/elseif/else语句，用法如下所示：

    
    #if( $foo < 10 )
        **Go North**
    #elseif( $foo == 10 )
        **Go East**
    #elseif( $bar == 6 )
        **Go South**
    #else
        **Go West**
    #end
  • 循环语句：

    • Velocity的循环使用的是foreach循环，如下所示：

    
        <ui>
            #foreach( $key in $allProducts.keySet() )
                <li>第${velocityCount}个元素为 ${key} ，其值为 ${allProducts.get($key)}</li>
            #end
        </ui>

    其中${velocityCount}是循环中的下标，从1开始。

    如果需要实现for(int i=0;i++;i<10)这样的效果，可以使用#foreach( $key in [0..9] )。

    循环中可以使用#break跳出循环。

  • 引入js和css文件：

    • 使用html标签引入js和css文件非常繁琐，但是使用Velocity可以将其变得更为便捷。使用方法如下：

    
        #set($jsfile = [jsfilePath])
        #js($jsfile)
        #set($cssfile = [cssfilePath])
        #css($cssfile)
  • 更多语法：

    • Velocity除以上常用语法外还有其他语法可供使用，详细可查看Velocity在Apache基金会中的引导页 Velocity - User Guide。
