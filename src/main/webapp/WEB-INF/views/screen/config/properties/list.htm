配置项 类型或取值范围 配置项说明1 配置项说明2
web.encoding String 项目的编码格式，默认为UTF-8
template.resource.loader.path String velocity的获取地址，默认为WEB-INF/views
velocity.onfigurerloader.path String velocity的properties获取地址
template.load.charset String velocity的模板编码格式，默认为UTF-8
system.devMode [true,false] 开发模式,默认为false,开启则启动一些日志,和调试选项.
velocity.velocimacro.library String velocity自定义marco的位置marco.vm文件，未配置则使用自带的文件
velocity.user.directive String velocity自定义命令(class)的位置
velocity.eventhandler.invalidreferences.class String Velocity处理无效参数的配置接口
velocity.eventhandler.referenceinsertion.class String velocity输出参数到页面前执行的接口，实现ReferenceInsertionEventHandler接口后配置在此处
velocity.eventhandler.nullset.class String Velocity处理空值的配置接口
velocity.layout.url String velocity的布局文件位置,默认为layout/default.htm
toolbox.xml.path String velocity的toolbox配置文件地址，默认为WEB-INF/conf/vm-toolbox.xml
web.vaild [true,false] 是否启用web校验,默认false
use.default.suffix.pattern [true,false] 使用默认的模版文件后缀，默认为true
melody.action.scan String controller的包扫描地址
velocity.in.jar.files String jar文件中的velocity模板文件位置
velocity.locale.country [true,false] 是否使用velocity模板的国际化配置,国家的配置，默认为true
velocity.locale.language [true,false] 是否使用velocity模板的国际化配置,语言，默认为true
velocity.locale.language.first [true,false] velocity模板的国际化配置,语言在前，默认为true
validation.message.bean.name String 校验消息类的名称，默认为空
performance.monitor.threshold Integer 行为监控的时间阈值，超过该阈值的request将会被记录下来，单位为毫秒，默认为100
session.max.inacterval Integer session的过期时间设置,单位为秒，默认为-1，即关闭浏览器session过期
session.meta.cookie.name String session在cookie中存储的名称，默认为 _h_
session.meta.cookie.locale.name String 用于存储locale信息的session在cookie中存储的名称，默认为 _l_
session.meta.cookie.domain String session在cookie中的域名如.baidu.com前面一定要有点，默认为app.server.host
cookies.max.length Integer cookie的最大长度，默认为4095
cookies.max.count Integer cookie的最大个数，默认为1
suportAsync.onoff [true,false] 开启异步加载，默认为true
app.server.host String appeserver的域名 appServer一般用来做服务器内部跳转,链接等
app.server.port Integer appServer的端口,默认80
app.server.protocol String appServer的协议,默认http
app.server.path String appServer的路径,默认/
app.server.followContextPath [true,false] appServer是否使用ContextPath，默认为false
image.server.host String imageServer的域名，默认为${app.server.host} imageServer用来获取静态文件,如js,css ,jpg,png等文件
image.server.port Integer imageServer的端口,默认80
image.server.protocol String imageServer的协议,默认http
image.server.path String imageServer的路径,默认/
image.server.followContextPath [true,false] imageServer是否使用ContextPath，默认为false
upload.server.host String uploadServer的域名，默认为${app.server.host} uploadServer用来获取上传的图片,或者指向上传服务器的地址
upload.server.port Integer uploadServer的端口,默认80
upload.server.protocol String uploadServer的协议,默认http
upload.server.path String uploadServer的路径,默认/
httpclient.max.total String httpclient的最大链接数，默认为200
httpclient.max.per.route Integer 每一个路由的最大连接数，默认为50
httpclient.socket.timeout Integer httpclient的超时时间，单位毫秒，默认3000
httpclient.connection.timeout Integer http连接的过期时间，单位毫秒，默认3000
httpclient.connection.request.timeout Integer http连接的每个请求过期时间，单位毫秒，默认5000
support.locales String[] 支持的地域，默认 zh_CN,en_US
default.locales String 默认地域,默认 zh_CN
default.locales String 默认地域,默认 zh_CN
system.negateDevMode Boolean devMode的相反值 该项不需要设置，默认为system.devMode取反
