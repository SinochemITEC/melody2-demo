Melody2新特性

1、更常用的文件后缀

Melody2中将页面文件的后缀更改为了".htm"，这样的后缀更具通用性，增强了兼容性。

2、新的Nosession用法

Melody2中使用Nosession不再需要创建CookieStore类了，只需要在xml中将相关信息配置好即可。

查看详细>>

3、新的拦截器

新版本使用了新的拦截器实现方法，需要继承HandlerInterceptorAdapter并实现HandlerInterceptor接口，该方法已经不区分url级别和方法级别，但是依然可以将处理粒度细化到方法上，具体实现实例见com.eyeieye.melody.demo.web.interceptor包中的拦截器。

查看详细>>

4、新的Resolver

Melody2中的Resolver不再是使用的WebArgumentResolver接口，而是使用的Spring自己的HandlerMethodArgumentResolver接口。
